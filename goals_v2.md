I want a single user MVP web app.

The development process should look like this:

1. Create a list of specifications for the app
2. Suggest a technical stack. The stack should be as simple as possible for a single user. For example, no need to have Docker.
3. Create a rolling-wave plan for implementation. The plan should allow me to test and approve the functionality and user interface at the end of each phase
